# 慢病用药小管家

帮助慢性病患者建立个人用药计划与复诊提醒的网页应用。

**在线访问：** `https://<你的用户名>.github.io/<仓库名>/`

---

## 功能

- **今日用药** — 按早/中/晚/睡前展示用药清单，一键打卡
- **我的药箱** — 管理药品信息、追踪库存余量、低库存预警
- **复诊计划** — 记录就诊信息（医院、医嘱、用药调整），自动推算下次复诊日期
- **用药记录** — 月历打卡视图、7天/30天依从率统计

数据存储在浏览器本地（localStorage），无需注册，无需服务器。

---

## 本地运行

直接双击 `index.html` 用浏览器打开，即可使用全部功能。

---

## 发布到 GitHub Pages（三步）

1. 将本仓库 push 到你的 GitHub
2. 进入仓库 **Settings → Pages**，Source 选择 `gh-pages` 分支，点击 Save
3. 等待约 1 分钟，访问 `https://<用户名>.github.io/<仓库名>/`

每次向 `main` 分支推送代码，GitHub Actions 会自动重新部署。
